Prerequisites

Hardware Requirement

  1. Processor: 1.60GHz or higher
  2. RAM: 3GB or higher

Software Requirement

  1. PHP 7 or higher
    • Install PHP on Ubuntu
    • Install PHP on Windows
    • Install PHP on Mac
  2. Composer 1.6.5 or higher
    • Install Composer on Ubuntu
    • Install Composer on Windows
    • Install Composer on Mac
  3. IMAP
    • Install IMAP on Ubuntu
    • Install IMAP on Windows
    • Install IMAP on Mac
  4. MailParse
    • Install MailParse on Ubuntu
    • Install MailParse on Windows
    • Install MailParse on Mac
  5. Apache Server
  6. MySQL Database
